## 课程学习内容

本课程是控制论的引入课程，课程内容远离网络安全，仅有工控网安全可能涉及相应知识，已于 2023 级退出网安必修全家桶。

课程内容包括：

- 控制论基本概念

- 开环控制系统和闭环控制系统

- PID 控制

- 专家控制
  
- 模糊控制

- 神经网络初步

- 基于神经网络的控制系统

- 智能算法
