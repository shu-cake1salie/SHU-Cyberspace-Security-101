## 课程学习内容

本课程主要讲授有关数据库技术和数据库原理的内容。

授课内容包括：

- 数据库系统简介

- 关系运算理论

- 模式设计理论

- 数据库语言

- 数据库设计

平时作业、实验主要以 SQL 语句和关系代数为主，要求能够写出相关陈述的 SQL 语句和关系代数表达式，以及两者之间的相互转换。

期末有大作业，由于 SHU 冬季学期中间有一个寒假，所以会把寒假时间纳入项目时间，整体量比较大，因而只要能做出来就有一个很高的分（具体内容见后）。
