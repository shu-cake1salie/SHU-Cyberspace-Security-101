## 课程学习内容

通过实验和教材学习自然语言处理的基本方法包括：分词、词法分析、句法分析、语义分析、语义关系抽取、语义角色标注等。

## 任课教师

=== "魏晓"

    - 课程内容主要通过自学和实验完成，老师会在课上讲解实验内容和要求，但是不会讲解实验的具体方法。
    - 上课以读PPT为主。
    - 每节课二维码签到
    - 作业是实验报告，每次实验都有一个报告，平时实验报告要求不高。
    - 期末报告是结合NLP功能做个小项目。
