## 课程学习内容

本课程主要讲述 C++ 语言语法及基本的算法设计

- C++ 的学习偏 C-style，有关类和对象的内容会在大二的面向对象程序设计继续讲授
- 算法设计需要在 SHU-OJ 平台完成对应练习，有 NOIP 基础会较轻松
